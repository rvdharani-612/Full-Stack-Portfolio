# Full Stack Portfolio Website

A modern Full Stack Portfolio Website built using the MERN Stack.

This project showcases:
- Personal portfolio information
- Skills section
- Projects section
- Contact section
- Dynamic project management using MongoDB
- Full frontend-backend integration

---

# 🚀 Tech Stack

## Frontend
- React.js
- CSS3
- Axios

## Backend
- Node.js
- Express.js

## Database
- MongoDB Atlas
- Mongoose

## Deployment
- Vercel (Frontend)
- Render (Backend)
